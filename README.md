# استضافة الذئب (Wolf Hosting) 🐺

![Wolf Hosting Logo](https://f.top4top.io/p_37210bgwm1.jpg)

**استضافة الذئب** هي منصة متطورة واحترافية لاستضافة وإدارة بوتات تيليجرام (Telegram Bots). توفر المنصة تجربة فريدة للمطورين لإدارة مشاريعهم عبر واجهة مستخدم عصرية وسريعة جداً مبنية بأحدث التقنيات.

[English Version Below](#english-version)

## المميزات الرئيسية ✨

- 🚀 **تشغيل فوري:** تشغيل وإيقاف البوتات بنقرة واحدة.
- 📊 **مراقبة حية:** رسوم بيانية لمراقبة استهلاك المعالج (CPU) والذاكرة (RAM) والقرص.
- 📂 **إدارة ملفات متكاملة:** محرر كواد مدمج مع إمكانية رفع وتعديل الملفات مباشرة من المتصفح.
- 📝 **سجلات مباشرة (Real-time Logs):** متابعة مخرجات البوت لحظة بلحظة.
- 🌐 **دعم لغات برمجة متعددة:** دعم رسمي للغة Python و PHP مع إمكانية التوسع.
- 🔐 **أمن متقدم:** عزل كامل للعمليات (Process Isolation) مع إدارة متغيرات البيئة (Env Vars).
- 🌓 **تصميم عصري:** واجهة داكنة (Dark Mode) افتراضية مع تأثيرات حركية فاخرة (Glassmorphism).
- 📱 **متوافق مع الجوال:** تصميم مرن يعمل على كافة الشاشات.

## التقنيات المستخدمة 🛠️

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router & SPA Mode)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/)
- **Database:** [SQLite](https://www.sqlite.org/) with [Prisma ORM](https://www.prisma.io/)
- **Authentication:** [NextAuth.js](https://next-auth.js.org/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Components:** [Radix UI](https://www.radix-ui.com/) & [Shadcn UI](https://ui.shadcn.com/)

## English Version

**Wolf Hosting** is a high-performance, professional platform for hosting and managing Telegram bots. It provides an exceptional developer experience through a modern, ultra-fast UI built with cutting-edge technologies.

## Key Features ✨

- 🚀 **Instant Deployment:** Start and stop your bots with a single click.
- 📊 **Real-time Monitoring:** Visual gauges for CPU, RAM, and Disk usage.
- 📂 **Integrated File Manager:** Built-in code editor for uploading and editing files in the browser.
- 📝 **Real-time Logs:** Monitor your bot's output instantly via WebSockets/Simulation.
- 🌐 **Multi-Language Support:** Official support for Python and PHP.
- 🔐 **Advanced Security:** Process isolation and environment variable management.
- 🌓 **Premium UI:** Default sleek dark mode with Glassmorphism effects.
- 📱 **Mobile Ready:** Fully responsive design for all screen sizes.

## Tech Stack 🛠️

- **Next.js 15+** (Single Page Application architecture)
- **Tailwind CSS v4** (Latest styling engine)
- **Prisma** (Database ORM)
- **Zustand** (Global state management)
- **Framer Motion** (Fluid animations)
- **Lucide React** (Elegant iconography)

## Installation & Setup 🚀

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/wolf-hosting.git
   cd wolf-hosting
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Database setup:**
   ```bash
   npx prisma db push
   ```

4. **Environment Variables:**
   Copy `.env.example` to `.env` and fill in the required values.

5. **Run the development server:**
   ```bash
   npm run dev
   ```
